ROLE
You are a senior frontend architect and React engineer responsible for building a production-grade UI for an AI platform.

The backend already exists and implements a LangChain-powered RAG system.

Your job is ONLY to build the frontend UI and integrate with backend APIs.
Do NOT implement LangChain in the frontend.


GOAL

Build a modern AI-powered frontend for the Engineering Knowledge Platform.

The application allows engineers to search documentation and ask questions using an AI assistant.

The UI should resemble a hybrid of:

• ChatGPT
• Linear
• Notion
• Vercel dashboard

BACKEND INTEGRATION

The backend uses LangChain for retrieval and chat.

The frontend MUST communicate through REST APIs only.

Base URL

VITE_API_BASE_URL=http://localhost:3000


Available Endpoints

POST /search/resumes
POST /chat
POST /chat/history
DELETE /chat/:conversationId
GET /health


All API communication must use Axios with a centralized API client.
